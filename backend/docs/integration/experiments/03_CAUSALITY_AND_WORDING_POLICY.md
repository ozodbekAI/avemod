# Causality And Wording Policy

Finance must distinguish:

- `before_after`: observed before/after change.
- `observational`: passive observation.
- `controlled_split`: only valid when real split assignment and variant metrics exist.

Before/after results must not be described as proven causal effects.

Required seller-facing disclaimer:

```text
Это наблюдаемая связь, а не доказанная причинность.
```

Allowed:

```text
Наблюдаемое улучшение: прибыль изменилась на 8%. Это наблюдаемая связь, а не доказанная причинность.
```

Forbidden:

```text
Фото принесло +120 000 ₽.
```

Confidence values are operational labels (`high`, `medium`, `low`) based on data completeness, sample size, confounders, and protocol adherence. They are not statistical probability.
